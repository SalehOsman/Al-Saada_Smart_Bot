# B-01: تسجيل بيانات — نموذج عام (Generic Data Entry)

> **الحالة:** ⏳ نموذج (يُبنى حسب احتياج الشركة)

## شجرة التدفق

```mermaid
flowchart TD
    A["الموظف يختار الموديول من القائمة\nمثال: ⛽ تسجيل الوقود"] --> B{"هل توجد مسودة؟"}
    B -->|نعم| B1["عرض: إكمال المسودة أم بدء جديد؟"]
    B -->|لا| C

    B1 -->|إكمال| C2["استعادة البيانات + متابعة"]
    B1 -->|بدء جديد| C

    C["الخطوة 1: إدخال الحقل الأول\nمثال: رقم لوحة السيارة"] --> D["🔍 validate()"]
    D --> D1{"صحيح؟"}
    D1 -->|لا| D2["⚠️ خطأ + إعادة"]
    D2 --> C
    D1 -->|نعم| E

    E["الخطوة 2: إدخال الحقل الثاني\nمثال: عدد اللترات"] --> F["🔍 validate()"]
    F --> F1{"صحيح؟"}
    F1 -->|لا| F2["⚠️ خطأ + إعادة"]
    F2 --> E
    F1 -->|نعم| G

    G["... باقي الحقول ..."] --> H["📋 شاشة التأكيد\nconfirm()"]
    H --> H1{"تأكيد؟"}
    H1 -->|لا| I["❌ إلغاء + حفظ مسودة"]
    H1 -->|نعم| J["💾 save()\n+ Audit + إشعار المدير"]
    J --> K["✅ تم حفظ السجل بنجاح"]

    style K fill:#27ae60,color:#fff
    style I fill:#e74c3c,color:#fff
```

## مثال: موديول تسجيل الوقود

| الحقل | النوع | تحقق |
|-------|------|------|
| رقم اللوحة | `string` | نمط لوحات مصرية |
| عدد اللترات | `number` | 1–999 |
| قراءة العداد | `number` | أكبر من آخر قراءة |
| ملاحظات | `string` | اختياري |
